This ruleset is based off of the original Korean TTRPG project developed by helloing0119. The second iteration was known only as the Community Rulebook, rebranded to the Community Rulebook 1.0.

This is the Community Rulebook 3.0, the fourth iteration of this game for western audiences and fans of the Project Moon franchise.
